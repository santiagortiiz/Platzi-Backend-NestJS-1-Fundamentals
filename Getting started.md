# Requirements
- ESLing
- Prettier code formatter
- Editor Config for VSCode
- npm i -g @nest/cli

# Guide
nest new <project_name>         # Create the project
npm run start                   # Start the project
npm run start:dev               # Start the project in live mode
http://localhost:3000/
nestjs use typescript by default
